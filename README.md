# Todo App - JavaScript CRUD + LocalStorage

A web application built with vanilla JavaScript that enables creating, editing, and deleting tasks with browser-based local storage persistence.

## Features

- ✅ Create tasks
- ✅ Edit tasks
- ✅ Delete tasks
- ✅ LocalStorage persistence
- ✅ Confirmation modal
- ✅ Dynamic DOM generation
- ✅ Auto-generated unique IDs
- ✅ Clean, responsive design

## Concepts Applied

- DOM manipulation
- Event listeners
- Array methods (forEach, findIndex, unshift)
- Destructuring
- Template literals
- JSON.stringify() and JSON.parse()
- CRUD operations
- localStorage API

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)

## Project Status

In development as part of a FreeCodeCamp workshop. Progressive improvements to structure and user experience are ongoing.
